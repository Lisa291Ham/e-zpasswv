# Contributing to e-zpasswv

Thank you for your interest in contributing to **e-zpasswv**!  
We welcome community contributions to improve functionality, design, and documentation.

---

## Contribution Guidelines

- Use clear, descriptive commit messages
- Follow existing project structure and coding style
- Ensure code is properly formatted and tested
- Avoid adding sensitive or real E-ZPass account data

---

## How to Contribute

1. Fork the repository
2. Clone your fork locally
3. Create a new branch:
   ```bash
   git checkout -b feature/your-feature-name
